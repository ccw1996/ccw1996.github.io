树莓派连接摄像头

发表于 2020-10-18 Disqus：
本文字数： 439 阅读时长 ≈ 1 分钟

注：在写该博文时，使用 树莓派 4B 8GB 版本，Ubuntu MATE 20.04.1 Beta 2 - 13 August 2020。摄像头型号为官方的 Pi Camera V2

树莓派在使用官方的 Raspberry Pi OS 时，可以使用 raspi-config 直接使能摄像头，然后即可使用。但在 Ubuntu Mate 20 中，因为种种原因，并不包含 raspi-config 。同时，通过修改 apt 源之后，再启用 raspi-config 时，会提示

但是，的确是存在 start_x.elf 文件。

可以通过以下方法去使能官方摄像头，也就是通过 CSI 方式启用的摄像头：

  1. cd /boot/firmware
  2. vim config.txt
  3. 在最底下的位置添加一行 start_x=1
  4. 重启 raspberry-pi

重启之后，在命令行输入 raspivid -p 0,0,640,480 -t 0 应该就能够看到摄像头的实时画面了。
